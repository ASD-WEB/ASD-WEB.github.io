# 模板字符串

模板字符串：用反引号来定义

let a='我是普通字符串';
let b = `我是模板字符串`;
console.log(`我喜欢你${b}`);
console.log(`求和${1+1}`);

优点

  1. 可以当做普通字符串使用
  2. 不受空格、回车的影响
  3. 可以嵌入变量、表达式等 方法：${变量或表达式}
